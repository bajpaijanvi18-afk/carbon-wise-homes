# 🌱 EcoCalc – Carbon Emission Dashboard

EcoCalc is a full-stack web application that calculates and analyzes building carbon emissions based on user inputs. It provides real-time insights, historical tracking, and an interactive dashboard to help visualize and understand environmental impact.

---

## 🚀 Live Demo
https://carbon-wise-frontend-jacy.onrender.com/

---

## 📌 Features

- 🌍 Calculate building carbon emissions from input parameters  
- 📊 Interactive and responsive dashboard UI  
- 🕒 Real-time data updates using REST APIs  
- 📁 History tracking of previous calculations  
- ⚡ Optimized backend for fast API responses  
- ☁️ Fully deployed on cloud for global access

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ecocalc.git
cd ecocalc

2. Install backend dependencies
cd backend
npm install
npm start

3. Install frontend dependencies
cd frontend
npm install
npm run dev

🔗 API Endpoints
POST /calculate → Calculate carbon emissions
GET /history → Fetch previous records
POST /save → Save emission data
